# ECS WebApp Infrastructure - CloudFormation

## Overview

This repository contains the **complete AWS infrastructure code** for deploying a web application with **frontend and backend services** using **ECS Fargate**, **ALB**, and **Aurora Serverless PostgreSQL** in a **brand new AWS account**.  

All resources are created using **CloudFormation YAML templates**, with a **root stack** that orchestrates nested stacks.
